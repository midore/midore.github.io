OS X 10.8.3: ssh keys in keychain.app

2013-04-07

SSH | Git | MountainLion

新しい repogitory を作り、git push したらユーザー名とパスワードを尋ねられしまったので、
.git/config の url をgit@ で始まるパスに変更。

# url = https://github.com/midore/myGh.git
url = git@github.com:midore/myGh.git


いざ git push すると今度は
ssh 鍵のパスワードを keychain.app に尋ねられてしまった。keychain.app には既に github.com 用の ssh key が登録してあるのだが...
これから github で新しいレポジトリーを作成するたびに keychain.app に ssh 鍵を尋ねられてしまうのだろうか？
以前は、github の ssh 鍵を keychain.app にいちど登録しておけば、keychain.app から再び ssh 鍵を尋ねられることはなかった。

以前のようにするにはどうすればいいのかな。
