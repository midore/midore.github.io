MountainLion | Git | SSH

先日 の 件。

新しいリポジトリーを作らなくても git push しようとすると keychain.app　が github の ssh　key のパスワードをいちいち尋ねてくる！と判明。驚く。

git push する前に keychain.app のロックを解除しておけば尋ねられないようだが...。

keychain.app がロックされている時、解除するユーザーのパスワードを求めないでいきなり git のssh パスワードを求めてくるのはなぜか。この鍵束は壊れてるのかな...。
