ブラウザで github.com にログインして新しいリポジトリ
(midore.github.io | https://github.com/midore/midore.github.io)
を作成。すると

We recommend every repository include a README, LICENSE, and .gitignore.
Create a new repository on the command line

とのことなのでブラウザから README などを "click" してコミット。

ローカルで

$ mkdir midore-github-io
$ cd midore-github-io
$ touch README.md
$ git init
$ git add README.md
$ git commit -m "first commit"
$ git remote add origin https://github.com/midore/midore.github.io.git
$ git push -u origin master

% git push -u origin master
